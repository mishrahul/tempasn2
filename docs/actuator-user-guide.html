üìñWhat is Spring Boot Actuator?

Spring Boot Actuator provides production-ready features to help you monitor and manage your application. It exposes operational information about the running application ‚Äî health, metrics, info, etc.

üè• Health Monitoring

Check the health status of your application, database, disk space, and other components.

üìä Metrics Collection

‚ò∏Ô∏è Kubernetes Integration

üí°Usage Examples

1. Check Application Health

# Using curl curl http://localhost:8080/api/v1/actuator/health # Using wget wget -qO- http://localhost:8080/api/v1/actuator/health # Using httpie http http://localhost:8080/api/v1/actuator/health

2. Get Application Information

# Get app info in JSON format curl http://localhost:8080/api/v1/actuator/info | jq . # Get specific field curl http://localhost:8080/api/v1/actuator/info | jq .app.version

3. Monitor Metrics (Requires Authentication)

# Set your JWT token TOKEN="your_jwt_token_here" # List all available metrics curl -H "Authorization: Bearer $TOKEN" \ http://localhost:8080/api/v1/actuator/metrics # Get JVM memory usage curl -H "Authorization: Bearer $TOKEN" \ http://localhost:8080/api/v1/actuator/metrics/jvm.memory.used # Get HTTP request statistics curl -H "Authorization: Bearer $TOKEN" \ http://localhost:8080/api/v1/actuator/metrics/http.server.requests # Get database connection pool metrics curl -H "Authorization: Bearer $TOKEN" \ http://localhost:8080/api/v1/actuator/metrics/hikaricp.connections.active

4. Health Check Script

#!/bin/bash # health-check.sh HEALTH_URL="http://localhost:8080/api/v1/actuator/health" status=$(curl -s $HEALTH_URL | jq -r .status) if [ "$status" == "UP" ]; then echo "‚úÖ Application is healthy" exit 0 else echo "‚ùå Application is unhealthy: $status" exit 1 fi
